# Сайт-визитка - Резюме Зазыкин Артем Сергеевич

## Описание
Современный сайт-визитка в стиле Apple с использованием полупрозрачных слоев (glassmorphism), зеленых оттенков и морской волны.

## Структура проекта
- `index.html` - основной HTML файл
- `style.css` - стили с анимациями
- `script.js` - JavaScript для интерактивности
- `1.png` - фотография (нужно добавить)

## Установка
1. Поместите файл с фотографией `1.png` в папку проекта
2. Для галереи: замените пути к изображениям в `index.html` (строки с `src="1.png"` в секции галереи) на ваши фотографии
3. Откройте `index.html` в браузере

## Особенности
- ✅ Прямоугольник с фото и ФИО в шапке
- ✅ Галерея с 6 фотографиями (3 видны, остальные можно перелистать)
- ✅ Увеличение фото по клику в модальном окне
- ✅ Адаптивное мобильное меню (гамбургер)
- ✅ Ссылки на социальные сети (ВКонтакте, Instagram, LinkedIn)
- ✅ Расширенная информация о создании чат-бота в Jivo
- ✅ Раздел "О себе" с хобби и увлечениями
- ✅ Улучшенный фон с градиентами и анимацией
- ✅ Шрифты с засечками (Playfair Display, Cormorant Garamond) для 50% текста
- ✅ Анимации при загрузке и скролле
- ✅ Стиль Apple (glassmorphism)
- ✅ Цветовая схема: зеленый и морская волна
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Плавные переходы и эффекты

## Использование галереи
- Стрелки влево/вправо для перелистывания
- Клик по фото для увеличения
- Навигация стрелками клавиатуры в модальном окне
- Закрытие по Escape или клику вне изображения

## Браузеры
Рекомендуется использовать современные браузеры (Chrome, Firefox, Safari, Edge).
